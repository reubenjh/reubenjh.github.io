When programming, what are some of your triggers? What causes a disproportionate emotional response?

This has happened a lot while learning JavaScript. When I can't figure out how to do something that seems trivially simple, like match up iterations in a loop to the index of an array that it might be applying the code to, so you can use an if statement to only execute code on certain index numbers. I was trying to do that while building the minesweeper game this sprint, to automatically apply row and collumn values to the cells and use the index numbers to know when to increase the value being aplied to row, and decrease the value being aplied to collumn. I ended up counting to five on my hands, and trying to remember that that is index 4 in the loop, about a hundred times, and getting more triggered each time.

Another moment I get triggered programming is when I realise I've completely forgotten how to do something I knew by heart only weeks or days before.

How did you find the Siberian North Railroad exercise (Stop, Breathe, Notice, Reflect, Respond)?

I really enjoyed applying it to recent things in life that I've been disproportionately frustrated with. I look forward to using it in programming for the kind of triggers above, but moreso I want to reflect on the things in life that cause this, the things with a deeper emotional history etc where some serious growth can be done I'm sure, or I can at least find a little control over my reaction. Don't feed the monsters!
