HTML, CSS and DOM (part 2)

* What is grid based design?
A design philosophy where you create a grid structure dividing the page up into columns and rows to house all of your elements. This makes it easy to lay out our elements in a visually appealling way.

* What's all the hype about responsive webpages?
Web traffic isn't only from computers, a huge portion actually browse on mobile phones or tablets, and each of those devices have very different screen sizes. You don't want your webpage to only work for some of those screen sizes and looked smooshed or unreadable on some, so web designers need to make webpages 'responsive', meaning they will respond to the size of the screen viewing it and adjust css accordingly to create a suitable design. * What is semantic structure?
The structure of what each section means. It's tags like header, nav, section and footer, making the code easier for someone else to understand, and easier to target certain areas with css that occur across multiple pages like a navigation bar.
