HTML, CSS and DOM (part 2)

What is grid based design?

A design philosophy where you create a grid structure dividing the page up into columns and rows to house all of your elements. This makes it easy to lay out our elements in a visually appealling way.

What's all the hype about responsive webpages?

Web traffic isn't only from computers, a huge portion actually browse on mobile phones or tablets, and each of those devices have very different screen sizes. You don't want your webpage to only work for some of those screen sizes and looked smooshed or unreadable on some, so web designers need to make webpages 'responsive', meaning they will respond to the size of the screen viewing it and adjust css accordingly to create a suitable design. People have pretty short attention spans so you'll lose them quick if your website looks like crap on a phone and the header image is all weirdly zoomed in and the text is a ridiculous size and they can't see the links because they're hidden somewhere in virtual space where they'd have to horizontally scroll to find them.


What is semantic structure?

The structure of what each section means. It's tags like div, header, nav, section and footer, making the code easier for someone else to understand, and easier to target certain areas with css that occur across multiple pages like a navigation bar.
