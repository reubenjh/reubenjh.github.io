Explain the difference between accessing data from arrays and objects.

To access data in an array, you need to target it's 'index', or 'numerical position in the array'. These are represented by a number appended at the end of a variable in square brackets. Just to make life a bit more nonsensical, the count starts at 0, as opposed to 1, since computers start counting from 0. So the left-most item in an array is index 0, the next index 1 and so on and so on.

Objects, on the other hand, store their data in a more human way by organising it into 'properties'. To access an object's data you need only to refer to the property that you're concerned with .......

Explain what functions are and why they are useful.

Functions are a repeatable bit of code that you can run on some arguments. They're useful as you only need to write out the function you want your JavaScript to perform once, then do it by calling the name of the function at any time. The arguments that it performs the code in question on can change. You could write a function to convert Celcius temperature to Fahrenheit, then call the function with the Celcius in brackets and it will spit out the converted Fahrenheit for you.
