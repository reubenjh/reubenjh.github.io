Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

There are a bunch of functions and tools in JavaScript that use 'true' or 'false' to make decisions in code. These look like 'if', 'for' or 'while'. You provide them with conditions - e.g. 'if (x = )', they check to see if the conditions evaluate to 'true', and if so, they execute the code you've written in their following curly brackets. If the condition is 'false', the code in the curlies is ignored. Control flow is the concept of using them, and nesting these inside one another to make complex decisions, to direct the code that is to be executed depending on the variables present.

A better way to explain that would be to draw my own control flow code for getting up in the morning using if statements:

      Then if (weather = cold) {
            Put on warm clothes,
      } Otherwise put on normal clothes,
      Then if (coffeejar does not = empty) {
            Make a coffee,
      } Otherwise panic that there is no coffee,
} Otherwise stay in bed.

Loops in JavaScript are ways to go through a list of data and apply code to each item without having to actually write it out heaps of times.

Explain the difference between accessing data from arrays and objects.

To access data in an array, you need to target it's 'index', or 'numerical position in the array'. These are represented by a number appended at the end of a variable in square brackets. Just to make life a bit more nonsensical, the count starts at 0, as opposed to 1, since computers start counting from 0. So the left-most item in an array is index 0, the next index 1 and so on and so on. If an array, called myArray, consists of the numbers 1, 5, and 9, then myArray[0] = 1 and mArray[2] = 9.

Objects, on the other hand, store their data in a more human way by organising it into 'properties'. To access an object's data you need only to refer to the property that you're concerned with. If an object, called myObject, has the properties "legs": 4, and "eyes": 2, then myObject["legs"] = 4, and myObject["eyes"] = 2.

Explain what functions are and why they are useful.

Functions are a repeatable bit of code that you can run on some arguments. They're useful as you only need to write out the function you want your JavaScript to perform once, then do it by calling the name of the function at any time. The arguments that it performs the code in question on can change. You could write a function to convert Celcius temperature to Fahrenheit, then call the function with the Celcius in brackets and it will spit out the converted Fahrenheit for you.
