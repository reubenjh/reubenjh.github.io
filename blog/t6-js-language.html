JavaScript Language

Explaining all the use cases of:

Parentheses: ()
  • Parentheses are used to surround the information that is passed to a function e.g. function myFunction (spoon, fork). When you call myFunction, you pass it the info representing the fork and the spoon e.g. myFunction ('airNZ branded fork', 'silver spoon') to perform its task on them. This is true for predefined functions too such as push() or addEventListener().
  • They're also used to define conditions or parameters in if, for or while loops. e.g. if (x === 10) {piece of code} will only execute the code if the condition that x strictly equals 10 in the parenthesis is true.
  • They can also be used to control the order in which calculations are done in more complicated maths problems. If a piece of arithmetic is in parenthesis, it will be calculated first. If there are parenthesis within those parenthesis, then those will be calculated first out of everything in the initial parenthesis. e.g. in let myAnswer = x * ((x + 2) / 2), JavaScript will see the x + 2 statement is in parenthesis nested in parenthesis and calculate that, before dividing by two and finally multiplying the product by 10 to give myAnswer.
Brackets: []
  • Brackets are used to access items in an array using the index, or to access properties of an object by using the property name e.g. myArray[2] selects the third item in myArray (since js counts starting from 0), and myObject['legs'] targets the property called 'legs' in the object.
